# Agent Skills profile

## Purpose

Apply one provider-neutral quality contract whenever an agent skill is created
or updated. The profile governs the reusable skill core; host-specific metadata
and installation remain explicit adapters.

## Start here

- `010_dset-implementation-agent-skills-specification-v1.md`
- `020_dset-implementation-agent-skills-profile-v1.toml`

## Boundary

This profile defines how skill packages are authored and assured. SKILL owns
DSET workflow semantics, TOOL owns deterministic validation and distribution,
and OPS owns release and hosted proof.
