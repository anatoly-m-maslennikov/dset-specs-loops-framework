# Version Scope — DSET 0.3 foundation

## Promise

Publish the public framework foundation, repository-local layered control
plane, Python CLI/package, reusable governance templates, thin skill wrappers,
and self-hosting baseline while stating remaining gaps honestly.

## Included

- Public methodology and documentation governance.
- Schema 1.2 layered project metadata and Work Areas.
- Dependency-light Python lifecycle, validation, traceability, bootstrap,
  archive, runtime checkpoint, skill distribution, and release foundations.
- GitHub delivery policy, CI definitions, supportability rules, provenance, and
  recorded proof history.

## Exclusions

- End-user adoption-readiness.
- Completed host-native Codex/Claude invocation proof.
- Complete cross-platform and hosted publication proof.
- Complete semantic-type, health, external-review, and conflict runtimes.
- JavaScript/TypeScript applied enforcement and external adopter proof.

claim only; it does not convert unfinished 0.4/0.5 work into a 0.3 promise.
