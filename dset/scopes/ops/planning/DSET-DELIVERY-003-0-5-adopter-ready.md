# Version Scope — DSET 0.5 adopter-ready

## Promise

Publish the first adopter-ready, multi-language DSET release with independently
repeatable cross-project proof.

## Included outcomes and capabilities

- JavaScript/TypeScript applied enforcement alongside Python and
  documentation/methodology profiles.
- Obsidian Your Harness as the owned external pilot.
- At least one additional clean repository adoption proving that the same
  skills resolve each project's local governance without policy leakage.
- Installation, migration, supportability, and release documentation sufficient
  for an operator who did not build the framework.

## Exclusions

- Stable `1.0` compatibility guarantees.
- Sales, customer-success work, performed production support, or operation of
  adopter deployment infrastructure.

## Exit criteria

- The completed 0.4 scope is the published baseline.
- Every declared language/profile and external adopter gate has fresh evidence.
- A Readiness Record explicitly accepts the exact 0.5 candidate.
