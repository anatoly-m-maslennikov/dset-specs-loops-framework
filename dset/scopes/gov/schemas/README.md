# GOV schemas

Repository-governance schemas owned by GOV:

- [Artifact areas](artifacts.schema.json)
- [Artifact types](artifact-types.schema.json)
- [Semantic atom frontmatter](atom.schema.json) — canonical typed relations plus
  sealed legacy `child_of` compatibility
- [Immutable atom ledger](atom-ledger.schema.json)
- [Append-only atom lifecycle](lifecycle.schema.json)
- [Changes](change.schema.json)
- [Governance registry](governance.schema.json) — schema 1.1 separates
  dependency from conflict precedence
- [Intake](intake.schema.json)
- [Provenance](provenance.schema.json)
- [Active authority compilation index](compilation.schema.json)
- [Governed conflict candidate](conflict-candidate.schema.json)
- [Governed conflict result](conflict-result.schema.json)
- [External review packet](review-packet.schema.json)
- [External review report](review-report.schema.json)
- [External review reconciliation](review-reconciliation.schema.json)

Other schema families live with their META, TOOL, SKILL, or OPS owner.
