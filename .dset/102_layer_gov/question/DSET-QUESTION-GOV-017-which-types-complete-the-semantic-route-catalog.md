# Question — Which Types complete the semantic route catalog?

Which single understandable canonical Type should occupy each of the 42
currently unnamed Revision-mode × Content-role × Governance-locus routes?

The resolution must:

- preserve the accepted seven Content roles and three Governance loci;
- assign exactly one Revision mode to every registered Type;
- leave no full route empty or multiply occupied;
- keep the current 21 role-and-locus names where their complete route remains
  correct;
- give every new Type one unique identity prefix;
- leave finer meanings to direct subtypes that inherit their parent Type's
  complete route; and
- remain understandable without requiring readers to decode the matrix first.

`DSET-QUESTION-GOV-013` separately owns the optional direct subtype vocabulary.
This Question owns only completion of the top-level route catalog.

## Rationale

The predecessor asked for 54 names under a six-role model. Assurance created a
seventh role, and the accepted 21-name matrix answered only the role-and-locus
surface rather than all 63 full routes.
