# Requirement — Outcome framing

Required effects belong to Definitions. Assessment criteria belong to Methods.
Measured results belong to Observations.

## Primary claim

An intended measurable state belongs in a Definition or assessment Method, while an observed result is an Observation; Outcome is not a routing axis or mandatory registered name.

## Rationale

Separating intended and observed state prevents shipped output from being mistaken for evidence that the intended effect occurred.
