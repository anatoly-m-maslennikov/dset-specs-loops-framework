# Deterministic methodology Tests

## Purpose

Provide the executable deterministic Test suite for the installed DSET
methodology.

## Boundaries

The synchronized `tests` package implements Test definitions owned by applied
QA atoms and plans. Test results and Verification conclusions remain applied
project evidence; they are never written into this methodology directory.

## Start here

- `010_dset-tool-tests-run.py` — location-independent installed-suite runner.
- `tests` — synchronized deterministic Test implementations.
