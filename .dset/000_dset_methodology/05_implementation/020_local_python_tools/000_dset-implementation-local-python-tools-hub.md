# Local Python Tools profile

## Purpose

Define the concrete `local-python-tools-v1` feature of the IMPL layer.

## Scope

Small command-line tools, scheduled utilities, repository automation, and
single-host Python workers. The profile excludes network services,
high-throughput multi-writer systems, public libraries, notebooks, and GUI
applications.
