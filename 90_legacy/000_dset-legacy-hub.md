# DSET legacy archive

## Purpose

Retain historical aggregate registries, completed migration records,
compatibility snapshots, pre-current change folders, and pre-current
documentation for deliberate investigation.

## Boundaries

This archive is inert, read-only history. It is outside the project-local
`.dset` control plane and is never an input to skill discovery, current
semantic compilation, project coverage, or executable governance.

Current authority exists only in `.dset`. Reintroducing archive content into
current truth requires a new governed artifact; copying an old carrier back is
not a valid restoration mechanism.

## Contents

- `aggregate-v1.3` — superseded multi-artifact registries.
- `framework` — earlier framework and self-hosting carriers.
- `methodology-todos` — completed or superseded methodology work notes.
- `migrations` — completed representation-migration records and snapshots.
- `versions` — superseded Version history and change folders.
